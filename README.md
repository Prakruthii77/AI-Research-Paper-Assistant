## Overview

AI Research Paper Assistant is an AI-powered web application built using Python and Streamlit to help researchers and students analyze academic papers efficiently.

The system performs:

- Keyword extraction  
- Research paper summarization  
- Citation extraction  
- Methodology explanation  
- Research insight generation  
- PDF export of results  

This project demonstrates the practical implementation of Natural Language Processing (NLP) techniques in academic research automation.

## How to Run

1. Clone the repository

git clone https://github.com/Prakruthii77/AI-Research-Paper-Assistant.git

2. Navigate to the project folder

cd AI-Research-Paper-Assistant

3. Install dependencies

pip install -r requirements.txt

4. Run the application

streamlit run app/streamlit_app.py
